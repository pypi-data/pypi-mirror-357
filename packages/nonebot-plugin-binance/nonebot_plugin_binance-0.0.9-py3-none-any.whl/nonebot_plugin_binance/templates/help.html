BN小助手 功能指南

核心指令

bn [币种]
一站式查询行情。默认查该币种对USDT在现货、U本位合约等市场的价格。
bn btc
bn eth-busd
bn k [币种] [周期]
获取多市场K线图。默认周期为`1d`。
bn k btc 1h
bn k eth
bn bal
查询全账户资产总览，包括现货、合约、杠杆等。
bn bal
bn pos
查看您在U本位和币本位合约的当前持仓。
bn pos
K线周期: m=分钟, h=小时, d=天, w=周, M=月. (例如: 15m, 4h, 1d)

账户管理

bn bind
【私聊】绑定您的API密钥。请在您绝对信任机器人提供方的情况下绑定！
bn bind [key] [secret]
bn unbind
解绑并删除您已存储的API密钥。
bn unbind
bn status
查询绑定状态和API密钥权限（现货、合约等）。
bn status
bn help
显示此帮助信息。
bn help

交易操作

bn open
查询所有市场的当前挂单 (可指定市场)。
bn open
bn open spot / um / cm
bn cancel
【暂不开放】为防止误操作，撤单功能默认关闭。
bn cancel ...
bn order
【暂不开放】为防止误操作，下单功能默认关闭。
bn order ...

价格预警

bn alert add
【群聊】添加价格预警。市场可选: `spot`(现货), `um`(U本位), `cm`(币本位)。
bn alert add um btc-usdt >90000
bn alert add btc-usdt <80000 (默认现货)
bn alert list
查看我设置的所有预警。
bn alert list
bn alert remove
移除指定ID的价格预警。
bn alert remove [预警ID]
