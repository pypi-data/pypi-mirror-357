VELRECOVER EXAMPLE FILES

This directory contains example files to help you get started with the velrecover application.

Contents:
- SEGY/: Contains sample SEGY seismic data files
- VELS/RAW/: Contains sample velocity data files in text format
- VELS/2D/: Directory where interpolated velocity models will be saved

Usage:
1. Load a velocity data file from VELS/RAW/ using the "Load Text File" button
2. Load a corresponding SEGY file from SEGY/ using the "Load SEGY File" button
3. Click "Interpolate" to generate the interpolated velocity model
4. Save the results to VELS/2D/ using the save buttons
